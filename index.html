基于Cent OS系统搭建CS:GO社区服务器

前期准备

云服务器（必须）：
本教程选用阿里云学生专享轻量应用服务器（包括5M峰值带宽、40GB SSD云盘、1000G流量包）安装CentOS 7.3系统作为服务器运行环境。

Xshell 7（可选）

WinSCP（可选）：
Windows环境下的SSH的开源图形化SFTP客户端，同时支持SCP协议，在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。

安装CS:GO服务器

以Root身份创建用户

useradd -m steam    // 添加名为steam的用户
passwd steam        // 为steam设置密码
cd home/steam/      // 切换到steam目录

[ls]可以查看当前目录的内容 [cd..]可以跳转到上一目录 [cd 文件夹名]可以打开文件夹

安装steamcmd

mkdir steamcmd      // 创建steamcmd文件夹
cd steamcmd/        // 切换到steamcmd目录
wget https://steamcdn-a.akamaihd.net/client/installer/steamcmd_linux.tar.gz    // 下载steamcmd
tar zxvf steamcmd_linux.tar.gz    // 解压
rm steamcmd_linux.tar.gz    // 删除压缩包

安装steamcmd运行所需要的工具和依赖库

sudo yum install glibc.i686
yum update libstdc++-4.8.5-11.el7.x86_64
yum install libstdc++.i686
./steamcmd.sh

出现”OK”表示安装完成

在steamcmd目录中安装csgo服务端

login anonymous     // 匿名登录steamcmd
force_install_dir ./csgo        // 创建CS:GO服务端安装文件夹
app_update 740 validate         // 安装CS:GO服务端

出现”Success! App ‘740’ fully installed.”表示安装完成
输入”quit”退出
