基于Cent OS系统搭建CS:GO社区服务器

本教程选用阿里云学生专享轻量应用服务器（包括5M峰值带宽、40GB SSD云盘、1000G流量包）安装CentOS 7.3系统作为服务器运行环境。

安装CS:GO服务器

以Root身份创建用户

useradd -m steam    // 添加名为steam的用户
passwd steam        // 为steam设置密码
cd home/steam/      // 切换到steam目录

[ls]可以查看当前目录的内容 [cd..]可以跳转到上一目录 [cd 文件夹名]可以打开文件夹

安装steamcmd

mkdir steamcmd      // 创建steamcmd文件夹
cd steamcmd/        // 切换到steamcmd目录
wget https://steamcdn-a.akamaihd.net/client/installer/steamcmd_linux.tar.gz    //下载steamcmd
tar zxvf steamcmd_linux.tar.gz    // 解压
rm steamcmd_linux.tar.gz    //删除压缩包
