定义：内部函数对外部作用域而非全局作用域对名字的引用，这样的内部函数被称为闭包函数

Python中怎样去创建一个闭包函数：

装饰器

本质上就是一个闭包函数

定义一个装饰器

def 装饰器名(func):
    def wrapper(*args, **kwargs):
        目标函数执行前，要添加的功能
        ret = func()
        目标函数执行后，要添加的功能
        return ret
    return wrapper

作用：在不改变源代码的情况下，给程序添加一个新的功能。

继承

多继承

· 定义：一个子类可以有多个父类，这就是多继承，并且子类拥有所用父类的熟悉和方法
· 语法：
class A:
    def test1(self):
        print('A类里是test1')
    def test2(self):
        print('B类里是test2')

class C:
    def test3(self):
        print('C类里是test3')

class B(A, C):
    def test(self):
        print('B类里是test')

# 实例化B的对象
b = B()
b.test()
b.test1()
b.test3()
· 注意事项：如果子类和父类有相同的方法，会优先调用子类的方法
· 小知识点：

  1. 在Python继承了object的类，都是新式类

  2. 在Python3+中，所有的类都是新式类

  3. 在Python2+中，继承了object的类是新式类，没有继承的是经典类

  4. 经典类目前在Python中的应用是没有的

  5. Python2+中经典类是按照深度优先去查找，新式类是按照广度优先去查找

  6. Python3+中所有的类都是按照广度优先查找

定义：在Python中类型具有特定的特殊功能的方法，在语法上表示为以下双下划线开头和双下划线结尾的形式，init() str()

销毁魔术方法：
  1. 当一个对象在内存中自动被销毁时，会自动执行；
  2. 参数至少要接收一个self对象；
  3. 没有返回值
class Cat:
    def __init__(self, name, age):
        self.name = name
        self.age = age
        print('&#123;&#125;来了，它今年&#123;&#125;岁了'.format(self.name, self.age))

    def __del__(self):
        print('&#123;&#125;走了'.format(self.name))

# 实例化对象
cat = Cat('John',4)
print('程序执行结束')

cat1 = Cat('Jame',6)
del cat1
print('程序执行结束')
