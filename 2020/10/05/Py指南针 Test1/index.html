基础

注释

单上注释：

- 格式：以 # 开头， # 号右边包含的内容都是

多行注释：

- 格式：以三对引号（单引号|双引号）英文状态下的引号 包含的内容

输入输出函数

输出函数

- print( “ 输出内容”)

输入函数

- input()
- 注意：input返回值类型为 str 字符串类型

数据类型

基本数据类型

字符串（ str ）

  • 格式：以引号包含的内容就是字符串

  • 引号：单引号，双引号，三引号

数值
  • 整数（ int ）
- age = 20
  • 浮点数（ float ）
- height = 1.80
  • 复数（ complex ）
- x= 5+12j
布尔类型( bool )

  • 注意: True | Fasle

  • 作用：判断事情的真假

空值（ None ）

组合数据类型

列表（ list ） [ 1, 2,3,4,5]
字典（ dict ）
元组（ tuple ）
集合（ set ）

变量：

变量：程序用来存储临时数据的容器

定义变量：
变量名 = 值
等号左边的内容就是变量名，等号右边的就是变量中保存的值
