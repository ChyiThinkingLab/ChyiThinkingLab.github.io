变量命名规范：

语法规范：变量是有数字、字母、下划线组成，且数字不能开头。
命名规范：

  • 见名知意：就是看到这个名字，大概就能了解到它是什么意思

数据类型转换

str – > int
a = '123'
print(type(a))  # <class 'str'>
# int
a1 = int(a)
print(type(a1))  #<class 'int'>
str –> float
b = "3.14"
b1 = float(b)
print(type(b1)) #  <class 'float'>
bool – > int | float
e1 = True
print(int(e1))
e2 = False
print(int(e2))
数值型转换成bool
d = 12
d1 = bool(d)
print(d1)
d2 = 0
d3 = bool(d2)
print(d3)
str — > bool
# 包含内容 返回 True
c = "12"
c1  =bool(c)
print(c1)  # True
# 不包含内容返回 False
c2 = ""
c3 = bool(c2)
print(c3)  # False
优先级：
  • bool < int < float
